28   behave is behavior-driven development, Python style.
29   
30   |logo|
31   
32   Behavior-driven development (or BDD) is an agile software development
33   technique that encourages collaboration between developers, QA and
34   non-technical or business participants in a software project.
35   
36   *behave* uses tests written in a natural language style, backed up by Python
37   code.
38   
39   First, `install *behave*.`_
40   
41   
42   Now make a directory called "features/".
43   In that directory create a file called "example.feature" containing:
44   
45   .. code-block:: gherkin 
46   
47       # -- FILE: features/example.feature
48       Feature: Showing off behave
49   
50         Scenario: Run a simple test
51           Given we have behave installed
52            When we implement 5 tests
53            Then behave will test them for us!
54   
55   Make a new directory called "features/steps/".
56   In that directory create a file called "example_steps.py" containing:
57   
58   .. code-block:: python 
59   
60       # -- FILE: features/steps/example_steps.py
61       from behave import given, when, then, step
62   
63       @given('we have behave installed')
64       def step_impl(context):
65           pass
66   
67       @when('we implement {number:d} tests')
68       def step_impl(context, number):  # -- NOTE: number is converted into integer
69           assert number > 1 or number == 0
70           context.tests_count = number
71   
72       @then('behave will test them for us!')
73       def step_impl(context):
74           assert context.failed is False
75           assert context.tests_count >= 0
76   
77   Run behave:
78   
79   .. code-block:: bash 
80   
81       $ behave
82       Feature: Showing off behave # features/example.feature:2
83   
84         Scenario: Run a simple test          # features/example.feature:4
85           Given we have behave installed     # features/steps/example_steps.py:4
86           When we implement 5 tests          # features/steps/example_steps.py:8
87           Then behave will test them for us! # features/steps/example_steps.py:13
88   
89       1 feature passed, 0 failed, 0 skipped
90       1 scenario passed, 0 failed, 0 skipped
91       3 steps passed, 0 failed, 0 skipped, 0 undefined
